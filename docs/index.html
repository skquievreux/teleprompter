Teleprompter

Eigenständige Desktop-App für Windows zum Vorlesen von Skripten während einer Video-Aufnahme. Kein Python nötig, einfach starten.

Installer (empfohlen) Portable .exe

Oder per Scoop: scoop bucket add skquievreux https://github.com/skquievreux/scoop-bucket && scoop install skquievreux/teleprompter
